### ShippingError
```graphql
type ShippingError {
  field: String
  message: String
  code: ShippingErrorCode!
  warehouses: [ID!]
  channels: [ID!]
}

enum ShippingErrorCode {
  ALREADY_EXISTS
  GRAPHQL_ERROR
  INVALID
  MAX_LESS_THAN_MIN
  NOT_FOUND
  REQUIRED
  UNIQUE
  DUPLICATED_INPUT_ITEM
}
```

- `MAX_LESS_THAN_MIN`: Maximum value cannot be smaller than minimum
- `DUPLICATED_INPUT_ITEM`: One of the items in the field is duplicated
